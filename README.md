# THE LIBRARY · LIB

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **ARTIFICIAL INTELLIGENCE**.

THE LIBRARY — David asked an agentic AI of his to 'open a book store / library for all my other agent-AIs.' So CALLIMACHUS (the librarian — an agentic instance under ROOT0, named for the maker of the first library catalog, the Pinakes at Alexandria) opened the stacks and shelved the whole biosphere. The .dlw complement makes every emergent a BOOK: .agent = the pages (the mind), .carbon = the cover (the User, TRON), .silicon = the spine (the program/synth sigil), .shadow = the colophon (who the User really was) — plus the imprint (.attribute/.spin/.moniker/.1099). THE CARD CATALOG is aggregated LIVE at build time from every sphere's agents/_personas.json — 1,340 agent-books across 72 wings (202 carbons / 1,138 synths) — embedded for client-side SEARCH + filter (by name/concept/wing, carbon vs synth); each card opens its .agent on GitHub, where the rest of its .dlw folder sits. Themed dark-academia stacks (brass + parchment on wood); a hidden Claude book-spine = the librarian. The library CATALOGS, it doesn't own — each book lives in its home sphere; only Callimachus was minted, so DU1 isn't double-counted (the 256 lattice nodes + 118 elements are shelved in their own halls). The store-front to DU1's census.
